# Typora 笔记索引生成器 - 接口设计说明书

## 1. 接口概述

### 1.1 接口设计原则
- **简洁性**：接口参数尽量精简
- **一致性**：命名风格统一
- **可扩展性**：便于未来功能扩展
- **向后兼容**：接口变更时保持向后兼容

### 1.2 接口列表
| 模块 | 接口 | 类型 | 说明 |
|------|------|------|------|
| `update_index.py` | `generate_md_index()` | 函数 | 主程序入口 |
| `update_index.py` | `get_file_icon()` | 函数 | 获取文件图标 |
| `update_index.py` | `build_folder_tree()` | 函数 | 构建文件夹树 |
| `update_index.py` | `build_tree_dict()` | 函数 | 构建树状字典 |
| `update_index.py` | `render_tree_dict()` | 函数 | 渲染 ASCII 树状图 |
| `update_index.py` | `render_folder_tree_simple()` | 函数 | 渲染目录列表 |
| `update_index.py` | `render_sub_content()` | 函数 | 渲染子内容 |
| `monitor.py` | `IndexHandler` | 类 | 文件事件处理器 |

## 2. 主程序接口

### 2.1 generate_md_index()

**函数签名**：
```python
def generate_md_index(base_dir: str = ".") -> None
```

**功能说明**：
生成 Markdown 格式的索引文件，包括文件夹结构图和目录列表。

**参数说明**：
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `base_dir` | `str` | `"."` | 笔记根目录路径 |

**返回值**：无（直接生成文件）

**使用示例**：
```python
# 生成当前目录索引
generate_md_index(".")

# 生成指定目录索引
generate_md_index("D:/我的笔记")
```

**内部调用关系**：
```
generate_md_index()
├── build_folder_tree()      # 构建目录树
├── build_tree_dict()       # 构建树状字典
├── render_tree_dict()      # 渲染树状图
├── render_folder_tree_simple()  # 渲染目录列表
│   └── render_sub_content()     # 递归渲染子内容
└── write file              # 写入输出文件
```

## 3. 辅助函数接口

```python
def get_file_icon(filename: str) -> str
```

**功能说明**：

**参数说明**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `filename` | `str` | 文件名或文件路径 |

**返回值**：emoji 图标字符串

**图标映射**：
```python
ICON_MAP = {
```

**使用示例**：

```python
icon = get_file_icon("README.md")     # 返回 "📝"
icon = get_file_icon("报告.docx")      # 返回 "📘"
icon = get_file_icon("data.csv")       # 返回 "📈"
```

### 3.2 build_tree_dict()

**函数签名**：
```python
def build_tree_dict(base_path: Path, md_files: list) -> dict
```

**功能说明**：
构建用于渲染 ASCII 树状图的字典结构。

**参数说明**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `base_path` | `Path` | 根目录路径对象 |
| `md_files` | `list` | 文件 Path 对象列表 |

**返回值**：嵌套字典结构（值为 `None` 表示文件）

**数据结构**：
```python
{
    "目录名": {
        "子目录": {...},
        "文件名.md": None
    }
}
```

**使用示例**：
```python
from pathlib import Path
base = Path("D:/笔记")
files = [Path("D:/笔记/编程/Python.md"), Path("D:/笔记/编程/Java.md")]
tree = build_tree_dict(base, files)
# 结果：
# {
#     "编程": {
#         "Python.md": None,
#         "Java.md": None
#     }
# }
```

### 3.3 render_tree_dict()

**函数签名**：
```python
def render_tree_dict(tree: dict, prefix: str = "", is_last: bool = True) -> list
```

**功能说明**：
递归渲染 ASCII 树状图。

**参数说明**：
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `tree` | `dict` | - | 树状字典结构 |
| `prefix` | `str` | `""` | 前缀字符串（用于递归） |
| `is_last` | `bool` | `True` | 是否为父级的最后一项 |

**返回值**：行字符串列表

**使用示例**：
```python
tree = {
    "文件夹": {
        "子文件.md": None
    }
}
lines = render_tree_dict(tree)
# 结果：
# ["├── 📁 文件夹/", "│   └── 📝 子文件.md"]
```

### 3.4 build_folder_tree()

**函数签名**：
```python
def build_folder_tree(files: list, base_path: Path) -> dict
```

**功能说明**：
构建用于渲染目录列表的字典结构，保留文件对象引用。

**参数说明**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `files` | `list` | 文件 Path 对象列表 |
| `base_path` | `Path` | 根目录路径对象 |

**返回值**：嵌套字典结构（值为 `Path` 对象表示文件）

**数据结构**：
```python
{
    "目录名": {
        "子目录": {...},
        "文件名.md": Path("D:/笔记/文件名.md")
    }
}
```

**与 build_tree_dict 的区别**：
| 函数 | 文件节点值 | 用途 |
|------|-----------|------|
| `build_tree_dict` | `None` | 仅用于渲染树状图 |
| `build_folder_tree` | `Path` 对象 | 用于生成链接和元信息 |

### 3.5 render_folder_tree_simple()

**函数签名**：
```python
def render_folder_tree_simple(tree: dict) -> list
```

**功能说明**：
渲染主文件夹列表（主入口函数）。

**参数说明**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `tree` | `dict` | 文件夹树字典 |

**返回值**：行字符串列表

**使用示例**：
```python
tree = build_folder_tree(files, base)
lines = render_folder_tree_simple(tree)
```

### 3.6 render_sub_content()

**函数签名**：
```python
def render_sub_content(tree: dict, depth: int = 0) -> list
```

**功能说明**：
递归渲染子文件夹和文件列表，使用树形缩进符号。

**参数说明**：
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `tree` | `dict` | - | 子树字典 |
| `depth` | `int` | `0` | 当前深度（从 0 开始） |

**返回值**：行字符串列表

**缩进规则**：
| 深度 | 符号 |
|------|------|
| 0 | 无前缀 |
| 1 | `├─` / `└─` |
| 2 | `│  ├─` / `│  └─` |
| 3 | `│  │  ├─` / `│  │  └─` |

## 4. 监控模块接口

### 4.1 IndexHandler 类

**类定义**：
```python
class IndexHandler(FileSystemEventHandler):
    """文件系统事件处理器，用于自动刷新索引"""
```

**继承关系**：
```
FileSystemEventHandler (watchdog)
    └── IndexHandler
```

**方法说明**：

#### on_any_event()

**方法签名**：
```python
def on_any_event(self, event: FileSystemEvent) -> None
```

**功能说明**：
处理任意文件系统事件，当检测到支持的文件变化时触发索引重建。

**参数说明**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `event` | `FileSystemEvent` | 文件系统事件对象 |

**触发条件**：
- 事件类型：`modified`、`created`、`deleted`、`moved`
- 文件扩展名在 `SCAN_EXTENSIONS` 中
- 不是目录变化

**使用示例**：
```python
from watchdog.observers import Observer

observer = Observer()
handler = IndexHandler()
observer.schedule(handler, path=".", recursive=True)
observer.start()
```
