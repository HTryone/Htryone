## 🌍 第一章：最基础的概念——认识浏览器和扩展

### 1. 什么是浏览器？

浏览器就是你用来“看网页”的那个软件。

比如你的电脑桌面上有一个蓝色字母 **e** 的图标，叫 **Microsoft Edge**；或者一个彩色圆形图标，叫 **Google Chrome**。它们就是浏览器。

<img src=.\p\Google.png alt="google" style="zoom: 40%; float: left;" /><img src=.\p\edge.webp alt="edge" style="zoom: 50%;" />

### 2. 什么是“扩展”？它和“插件”是一回事吗？

- **扩展（Extension）**：可以理解成**给浏览器安装的小工具App**。
- **插件（Plugin）**：这是老一辈的叫法，现在我们常说的“浏览器插件”，**99%指的都是扩展**。

**举个例子帮你理解：**

| 类比对象       | 说明                                                         |
| :------------- | :----------------------------------------------------------- |
| 你的手机       | 刚买来只能打电话、发短信、拍照                               |
| 下载的App      | 微信、支付宝、美颜相机，让手机功能变多                       |
| **浏览器扩展** | 给浏览器安装的小工具，让浏览器能**挡广告、翻译网页、调视频速度** |

**一句话总结：扩展就是浏览器的“外挂技能包”。**

### 3. 在哪能找到这些扩展？

Edge浏览器有自己的“应用商店”，叫做 **Microsoft Edge 加载项**。

进入方法：点浏览器右上角的 **三个小点（···）** → 选择 **“扩展”**，就能看到商店入口。

## ⚙️ 第二章：先把Edge浏览器本身调顺了

装扩展之前，我们先把浏览器的“系统设置”调一下。就像搬新家先打扫卫生一样，做完这几步，浏览器会更清爽、更保护隐私。

### 怎么打开设置？

点右上角 **三个小点（···）** → 选择 **“设置”**。

### 建议调整的项目

| 设置路径                                                  | 怎么做                 | 有什么用                                   |
| :-------------------------------------------------------- | :--------------------- | :----------------------------------------- |
| 系统和性能 → 启动增强                                     | **关掉**               | 不让浏览器一开电脑就偷偷启动，省内存       |
| 系统和性能 → 在Microsoft Edge关闭后继续运行后台扩展和应用 | **关掉**               | 关掉浏览器后所有后台进程都停掉，电脑更流畅 |
| 隐私、搜索和服务 → 个性化广告                             | **关掉**               | 减少广告根据你的兴趣精准推送               |
| 隐私、搜索和服务 → 安全性→增强web安全性                   | **关掉**               | 减少插件兼容性                             |
| 启动、主页和新标签页 → 新标签页设置                       | 把不用的栏目都**关掉** | 打开新标签页时干干净净，不眼花             |

调一次就行，以后不用再管。

## 🧩 第三章：手把手教你安装扩展

### 方法一：从官方商店在线安装（最推荐）

1. 点击浏览器右上角长得像 **拼图块** 的图标（如果没有这个图标别急，**第六章会教你怎么找回来**）。

2. 点击 **“获取 Microsoft Edge 扩展”**。<img src=.\p\1.png alt="edge" style="zoom:35%;float:right" />

3. 在搜索框输入想要找的扩展名字。

4. 找到后点击 **“获取”**，就自动装上了。

   <img src="D:\desktop\浏览器指南\p\2.png" alt="2" style="zoom:30%;float:left" />

   

装好以后，拼图块旁边会出现新扩展的小图标，以后点它就能用。

### 方法二：离线安装（当商店打不开时）

有些时候网络不好商店进不去，就需要用离线安装。你会拿到两种文件：`.crx` 和 `.zip`。

#### 情况A：你下载的是 `.crx` 文件

1. 在浏览器地址栏输入 `edge://extensions/` 然后回车。
2. 打开页面右上角的 **“开发者模式”** 开关。
3. 把 `.crx` 文件**直接拖进**这个页面里，松手，就装好了。

#### 情况B：你下载的是 `.zip` 压缩包

1. 先对 `.zip` 文件**解压**（右键点文件 → “全部解压缩”），得到一个文件夹。
2. 同样进入 `edge://extensions/` 页面，打开 **“开发者模式”**。
3. 点击左上角的 **“加载解压缩的扩展”**。
4. 在弹出的窗口里，找到解压出来的文件夹，**双击点进去**，选择里面的**根文件夹/ `mv-3` 文件夹**（如果没有就选最里面那一层），然后点“选择文件夹”就完成了。

## ⚠️ 第五章：安全警示！这些坑一定要避开

扩展虽好用，但坏人也可能制作“有毒”的扩展来偷你的密码和浏览记录。以下规则务必遵守。

### 四条安全铁律

1. **只在官方商店安装**。Edge商店、Chrome商店都是官方的，别去不知名网站下载。
2. **安装前看权限**。如果它要求“读取所有网页数据”，而它只是个计算器，那就有鬼。
3. **看下载量和评分**。用的人多、评分高的通常靠谱。
4. **不用的扩展赶紧删**。扩展多了电脑会慢，而且留着也是安全隐患。

### 🚫 已被曝光的恶意扩展黑名单

以下扩展已被安全机构证实会窃取用户数据。如果你装了，请立刻卸载！

- **WeTab 新标签页**
- **Infinity V+ 新标签页**
- **Clean Master**
- **Phantom Shuttle**

**卸载方法**：点右上角拼图块图标 → 找到这些扩展 → 点旁边的三个点 → 选择“从 Microsoft Edge 中删除”。

## 🔍 第六章：找不到“拼图块”扩展图标怎么办？

有时候刚装好浏览器，扩展图标不一定在地址栏旁边显示。别慌，按下面方法找回来。

### 方法1：直接进后台（最快）

在地址栏输入 `edge://extensions/`（Edge浏览器）或 `chrome://extensions/`（Chrome浏览器）并回车，就能看到所有扩展。

### 方法2：快捷键唤醒

按键盘上的 **`Ctrl + Shift + E`** 快捷键，扩展管理面板会立刻弹出来。

### 方法3：从菜单里找

1. 点右上角 **三个小点（...）**。
2. 在菜单里找到 **“扩展”** 选项（有时候它的图标就是个拼图块）。
3. 点击它，扩展列表就出来了。

### 怎么让图标一直显示在工具栏上？

1. 进入 `edge://extensions/` 页面。
2. 确保你要用的扩展是**开启状态**（开关是蓝色的）。
3. 点击扩展卡片上的 **“详细信息”**。
4. 找到 **“在工具栏中显示”** 选项，把开关打开。或者直接在扩展列表弹窗里，点扩展旁边的 **图钉** 图标，把它“钉”在工具栏上。

## ⚙️ 第七章：其他实用技巧（Edge 和 Chrome 通用）

### 1. 如何更换浏览器的默认搜索引擎？

地址栏既是输入网址的地方，也是搜索框。你可以自己决定用哪个搜索引擎（百度、必应、谷歌等）。

**Edge 浏览器操作步骤：**
1. 点右上角三个点 → **设置**。
2. 左侧菜单选择 **“隐私、搜索和服务”**。
3. 滚动到最底部，点击 **“地址栏和搜索”**。
4. 在“地址栏中使用的搜索引擎”旁边，点击下拉菜单选择你想要的。

**Chrome 浏览器操作步骤：**
1. 点右上角三个点 → **设置**。

2. 左侧菜单选择 **“搜索引擎”**。

   > **谷歌浏览器国内环境下推荐使用Bing引擎，否则无法使用**

3. 点击“地址栏中使用的搜索引擎”旁边的下拉菜单进行选择。

### 2. 如何把你喜欢的浏览器设为“默认浏览器”？

如果电脑总是用你不熟悉的浏览器打开网页，可以把你常用的设为默认。

**Windows 电脑：**
- **Edge浏览器**：点三个点 → 设置 → 左侧选“默认浏览器” → 点击“设为默认值”。
- **Chrome浏览器**：点三个点 → 设置 → 左侧选“默认浏览器” → 点击“设为默认浏览器”。

**Mac 电脑：**
- 点屏幕左上角苹果图标 → **系统设置** → **桌面与程序坞**。
- 找到“默认网页浏览器”，在下拉菜单里选你常用的那个。
